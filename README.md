## âŸ¡ Overview
**YOUniverse** is a console-based Java application designed to help users explore their personality type through a modernized, Gen-Z friendly assessment engine.

Unlike standard static quizzes, YOUniverse offers an interactive CLI (Command Line Interface) experience with:
* **Visual Trait Mapping:** Custom-built ASCII bar charts to visualize introversion/extroversion and other traits.
* **Personalized Feedback:** Dynamic naming and customized advice based on the 16-personality framework.
* **Object-Oriented Design:** Practical implementation of OOP principles like Inheritance, Polymorphism, and Encapsulation.
